# brcatcga
tcga data from breats cancer patients (clinical, mutational, RNA, methylations)

MLcorrelations7.csv is the file used to train the breast cancer algorithm with multi-omics data.
MLcorrelationsonlymut.csv is the file used to train the breast cancer algorithm with mutation data.
gliomaML.csv is the file used to test the breast cancer algorithm with multi-omics data on a glioma dataset.
ovarianML.csv is the file used to test the breast cancer algorithm with multi-omics data on a ovarian cancer dataset.
uterineML.csv is the file used to test the breast cancer algorithm with multi-omics data on a uterine cancer dataset.

survivalbrcaV2.ipynb is the jupyter notebook used to train the breast cancer algorithm with multi-omics data and plot performance metrics as well as SHAP values.
survivalbrcamutationalV2.ipynb is the jupyter notebook used to train the breast cancer algorithm with mutation data and plot performance metrics as well as SHAP values.
