# brcatcga
TCGA data from breats cancer patients (clinical, mutational, RNA, methylations) (data release 43, GRCh38/hg38). The results presented here are in whole or part based upon data generated by the TCGA Research Network: https://www.cancer.gov/tcga.

MLcorrelations7.csv is the file used to train the breast cancer algorithm with multi-omics data and the Lasso Cox method.
MLcorrelationsonlymut.csv is the file used to train the breast cancer algorithm with mutation data.
MLbasedOnRSF.csv is the file used to train the breast cancer algorithm with multi-omics data and the RSF feature importance methods.
survivalbrca.ipynb is the jupyter notebook used to train the breast cancer algorithm with multi-omics data and plot performance metrics as well as SHAP values.
